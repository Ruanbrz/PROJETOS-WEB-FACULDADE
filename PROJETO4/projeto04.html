CSS e JavaScript

O que é o arquivo CSS?

CSS é a sigla para Cascading Style Sheets, em outras palavras, Folhas de Estilo em Cascata. Esse arquivo é usado para para personalizar o visual de um site, ou seja, deixa o site com uma apresentação mais amigável para o usuário. O arquivo CSS complementa o HTML. Normalmente as configurações CSS, que estão dentro do marcador (style) ficam em um arquivo separado do arquivo HTML, isso trás alguns benefícios, como: fornecer uma estrutura mais limpa; possibilitar uma melhor manutenção; o mesmo arquivo de extensão .CSS pode ser usado por várias páginas; dividir o projeto em camadas específicas deixando mais claro o projeto.

Recomendações

Use sempre endentação (ou identação ou indentação) no código HTML, CSS e JS. A endentação ajuda a manter a hierarquia visual do código facilitando a leitura do mesmo. Separe logicamente as categorias de estilo no arquivo CSS, use comentários corretamente de modo a explicitar essas partes (corpo da página, listas, tabelas, rodapé, cabeçalho etc.). Crie, junto a equipe de desenvolvimento, um padrão para nomes de classes, IDs, seletores e variáveis. Usar padrões de nomenclatura auxiliará a documentação, manutenção e entendimento do código.

JavaScript

É uma linguagem de script para WEB, criada pela Netscape (Brendan Eich) em 1994. Trata-se de uma linguagem de front-end, usada por bilhões de páginas. A linguagem JavaScript tem por objetivo adicionar funcionalidades às páginas HTML, ela é usada dentro das páginas HTML.
